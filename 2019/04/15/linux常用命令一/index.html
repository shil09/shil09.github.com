发表于 2019-04-15

linux常用命令

一、文件和目录权限

  1. chmod == change mode
    -rwxr–r– 744
    -:文件类型
    rwx:读、写、执行
    分为三段：文件所有者、本组其他用户、其他组用户
  2. chmod 777 shilei
    给shilei目录授予777权限
  3. chmod -R 777 shilei
    给shilei目录机器子目录授予777权限
  4. chmod u=rwx,g=r,o=r,shilei :一种写法

二、更改所有者和所属组

  1. chown
    chown stone:stone 1.txt:把1.txt的所有者和所属组改为stone.
  2. chgrp
    chgrp stone 2.txt :把2.txt的所属组改为stone.
  3. chown :root 3.txt ：仅仅改3.txt的所属组
  4. chown -R stone:stone /root/shilei
    :修改shilei目录及其子目录所有者为stone，所属组为stone。
    语法：chown -R username:group filename

    三、umask

  5. umask值用于设置用户在创建文件时的默认权限，当我们在系统中创建目录或文件时，目录或文件所具有的默认权限就是由umask值决定的。
    对于root用户，系统默认的umask值是0022；对于普通用户，系统默认的umask值是0002。执行umask命令可以查看当前用户的umask值。
    `
    [root@localhost ~]# umask

0022

umask值一共有4组数字，其中第1组数字用于定义特殊权限，我们一般不予考虑，与一般权限有关的是后3组数字。

默认情况下，对于目录，用户所能拥有的最大权限是777；对于文件，用户所能拥有的最大权限是目录的最大权限去掉执行权限，即666。因为x执行权限对于目录是必须的，没有执行权限就无法进入目录，而对于文件则不必默认赋予x执行权限。

对于root用户，他的umask值是022。当root用户创建目录时，默认的权限就是用最大权限777去掉相应位置的umask值权限，即对于所有者不必去掉任何权限，对于所属组要去掉w权限，对于其他用户也要去掉w权限，所以目录的默认权限就是755；当root用户创建文件时，默认的权限则是用最大权限666去掉相应位置的umask值，即文件的默认权限是644
`

四、隐藏权限chattr&lsattr

  1. chattr +i 1.txt :设置隐藏权限
  2. lsattr 1.txt ：显示隐藏权限
  3. 删除隐藏权限： chattr -i 1.txt

    五、特殊权限set_uid

  4. ls -l /usr/bin/passwd
  5. chmod u+s /bin/ls
    给其他用户授予ls的root权限
  6. chmod u-s /bin/ls
    去除其他用户ls的root权限
  7. 给普通用户执行该命令所有者的权限

六、特殊权限set_gid & stick_bit

  1. 普通用户拥有所属组的权限
  2. chmod g+s shilei
  3. stick_bit :防止其他用户删除权限，所以说，这个stick_bit权限叫防删除位，只有有权限的用户（比如root）才可以删除，而其他的用户是根本无法删除掉的
