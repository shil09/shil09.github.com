linux变量

一、变量
1.定义

#a=1

#echo $a
2.列出变量

#env(列出全局变量)

#export a=1(声明全局变量)

#set 可以列出所有变量

#a=1,不能有空格，变量名大小写字母，数字，下划线，但是不能以数字开头

#a=’shilei’，一般用单引号

3.反引号
表示一条命令

#a=3

#b=echo $a

3

#b=’echo $a’

#echo $b

echo $a //单引号表示字符串

#b=”echo $a”

#echo $b
echo 3 //双引号不严谨，会解析a

二、环境变量
1.设定与取消

#export a=1;

#bash

#echo $a

1

#exit

#unset a

#bash

#echo $a

#exit
//全局变量——所有shell

//局部变量，子shell

2./etc/profile /etc/bashrc
全局变量,系统配置文件，针对所有用户
~/.bashrc ~/.bash_profile家目录，针对用户自己
~/.bashrc用户登录时执行,然后再执行.bash_profile；

  1. 在shell中， . == source,表示执行；#表示注释

三、特殊符号
1、反斜杠\,
2、$表示变量
3、表示命令
4、>重定向
5、 &&连接两个命令
6、 &把命令放到后台执行
7、 cut命令：分段

#cut -d ‘:’ -f 1 /etc/passwd
说明：-d后面是分隔符：，-f 1表示截取第几个字段，/etc/passwd表示文件

#cut -d ‘:’ -f 1-3 /etc/passwd
说明：-d后面是分隔符：，-f 1-3表示截取第1到3个字段；1和3个字段表示为1,3，/etc/passwd表示文件
8、 sort命令

#sort -t ‘:’ -k3 /etc/passwd
:-t表示分隔符，-k3表示以第三个排序(ASCII码)，十进制排序sort -t ‘:’ -k3 -n /etc/passwd

  1. 排序后截取某一字段
    #sort -t ‘:’ -k3 /etc/passwd | cut -d ‘:’ -f 3
    #sort -nru | -c :n表示十进制，r表示倒序,u表示重复的只显示一个 -c表示统计次数；
    10.wc命令
    #wc /etc/passwd
    24 54 1313 /etc/passwd
    24行，54个单词，1313个字节；
    wc -l /etc/passwd:显示有多少行
    #du -sh 1.txt显示文件大小
  2. split -b 1M 1.txt
    把1.txt切为每份1M的大小
  3. 特殊符号
    &&: cd /tmp/ && ls 连接两个命令并执行，第一个命令执行成功时才去执行第二个命令。
    ||: cd /tpm/ || ls 第一个命令不成功时才去执行第二个命令。
    ； cd /tpm/ ； ls：表示两个命令没有关系，顺序执行，即使第一个执行失败第二个也会执行。
